## ⚠️ Important Notes

- **Active Development**: This server emulator is under active development. Many features are data-ready but gameplay systems are not yet fully implemented.
- **Current Functionality**: Authentication, character management (create/delete/select), server connectivity, and network routing are working.
- **Data Structures**: Extensive database records for pets, monsters, items, NPCs, and shops are defined but gameplay handlers are in progress.
- **Educational Purpose**: This server is for educational and preservation purposes.
- **Always backup** your database before updates.
- Check the [Troubleshooting](Troubleshooting) page for common issues
